# OWASP Top 10: Identification and Authentication Failures — Junior / Entry Level

1. What is authentication? — Verifying user identity.
2. What is identification? — Claiming an identity (username/email).
3. What is credential stuffing? — Using leaked creds from other sites.
4. What is brute force? — Guessing passwords repeatedly.
5. What is MFA? — Multiple factors to verify identity.
6. What is a weak password policy? — Short or common passwords allowed.
7. What is a password reset token? — A temporary token to reset passwords.
8. What is account enumeration? — Identifying valid accounts via responses.
9. What is session fixation? — Forcing a known session ID.
10. What is session timeout? — Expiration of sessions after inactivity.
11. What is secure cookie usage? — HttpOnly, Secure, SameSite flags.
12. What is a credential leak risk? — Exposed passwords or tokens.
13. What is a login rate limit? — Limiting login attempts.
14. What is a password hash? — One-way representation of a password.
15. What is a phishing risk? — Users tricked into giving credentials.
