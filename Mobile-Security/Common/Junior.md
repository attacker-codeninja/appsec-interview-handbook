# Mobile Application Security (Common) — Junior

1. What is OWASP MASVS? — Mobile app security verification standard.
2. What is OWASP MASTG? — Mobile app security testing guide.
3. Why is local storage risky? — Data can be extracted from the device.
4. What is certificate pinning? — Trusting specific certificates or public keys.
5. What is jailbreak/root detection? — Detecting compromised devices.
6. What is app tampering? — Modifying APK/IPA to change behavior.
7. What is reverse engineering? — Analyzing app binaries to understand logic.
8. Why are hardcoded secrets dangerous? — They can be extracted and abused.
9. What is secure storage on iOS and Android? — Keychain and Keystore.
10. What is an insecure mobile API? — Backend lacks authZ or input validation.
11. What is mobile app transport security? — Enforcing TLS and safe configs.
12. What is clipboard leakage? — Sensitive data left in clipboard.
13. Why is logging sensitive data bad? — Logs can be exfiltrated.
14. What is deep link risk? — Untrusted URI handling and hijacking.
15. What is mobile threat modeling? — Modeling device, app, and backend threats.
