# Infrastructure and Cloud Security (General) — Mid-Senior

1. How do you design least privilege IAM at scale? — Use roles, policies, and access reviews.
2. How do you secure cloud storage? — Block public access and encrypt.
3. How do you manage multi-account strategy? — Separate prod/dev with SCPs.
4. How do you detect misconfigurations? — CSPM and config rules.
5. How do you secure secrets in cloud? — KMS and secret managers.
6. How do you handle cross-account access? — Use roles and external IDs.
7. How do you secure serverless and containers in cloud? — IAM, network, and scanning.
8. How do you manage logging at scale? — Centralize with SIEM.
9. How do you handle cloud key rotation? — Automated rotation schedules.
10. How do you secure cloud APIs? — MFA, IP allowlists, and anomaly detection.
11. How do you secure data pipelines? — Encryption, access control, and validation.
12. How do you implement zero trust in cloud? — Identity-based access and segmentation.
13. How do you handle shadow IT? — Asset discovery and policy enforcement.
14. How do you audit cloud compliance? — Evidence collection and controls mapping.
15. How do you secure multi-region deployments? — Consistent policies and centralized logs.
