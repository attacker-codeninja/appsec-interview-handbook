# Secret Management — Junior

1. What is a secret? — Credentials or keys that grant access.
2. Why is hardcoding secrets bad? — They can be extracted and abused.
3. What is a secret manager? — Service for secure secret storage.
4. What is rotation? — Replacing secrets on a schedule.
5. What is least privilege for secrets? — Minimal scope and access.
6. What is a KMS? — Key Management Service.
7. What is an HSM? — Hardware security module for keys.
8. What is env var leakage? — Secrets exposed in process or logs.
9. What is a token TTL? — Time to live for a token.
10. What is a shared secret? — Same secret used by multiple parties.
11. What is secret sprawl? — Secrets spread across systems.
12. What is a breach response for secrets? — Rotate and revoke.
13. Why should secrets be audited? — Detect misuse and access.
14. What is secret scanning? — Finding secrets in repos.
15. What is static vs dynamic secret? — Fixed key vs on-demand credential.
