# OWASP Top 10: Broken Access Control — Junior / Entry Level

1. What is broken access control? — Missing or inconsistent authorization checks.
2. What is IDOR/BOLA? — Accessing another user's object by changing an ID.
3. What is BFLA? — Accessing functions outside your role or scope.
4. What is forced browsing? — Guessing or enumerating protected URLs.
5. What is vertical vs horizontal access control? — Role level vs same-role object access.
6. What is RBAC? — Role-based access control.
7. What is ABAC? — Attribute-based access control.
8. Why is client-side access control insufficient? — Client checks can be bypassed.
9. What is privilege escalation? — Gaining higher access than intended.
10. What is a secure cookie flag? — HttpOnly, Secure, SameSite settings.
11. What is session fixation? — Forcing a user into a known session ID.
12. What is session hijacking? — Stealing a session token.
13. What is an insecure direct object reference? — Direct access to objects without authZ.
14. Why are default admin accounts risky? — Known credentials and elevated privileges.
15. Why must authZ be enforced on the server? — The server is the source of truth.
