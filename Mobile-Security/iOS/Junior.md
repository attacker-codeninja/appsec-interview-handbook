# iOS Security — Junior

1. What is the iOS Keychain? — Secure storage for secrets.
2. What is App Transport Security (ATS)? — Enforces TLS policies.
3. What is a URL scheme risk? — Other apps can hijack custom schemes.
4. What is a Universal Link? — Secure link bound to your app domain.
5. What is jailbreaking? — Removing OS restrictions.
6. What is an IPA? — iOS app package file.
7. What is code signing? — Verifies app integrity and origin.
8. What is the Secure Enclave? — Hardware security for keys.
9. Why is plist storage risky? — Can store secrets in plaintext.
10. What is a Keychain access group? — Shared Keychain across apps.
11. What is the risk of UIWebView? — Deprecated, less secure than WKWebView.
12. What is iOS backup risk? — Sensitive data can be included in backups.
13. What is `NSAllowsArbitraryLoads`? — Disables ATS, increases MITM risk.
14. What is entitlements misuse? — Extra capabilities increase risk.
15. What is pasteboard risk? — Sensitive data leakage via clipboard.
