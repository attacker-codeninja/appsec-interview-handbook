# Android Security — Junior

1. What is an Android APK? — The package format for Android apps.
2. What is an Intent? — A messaging object between components.
3. What is an exported component? — A component accessible to other apps.
4. Why are broad permissions risky? — Increases attack surface.
5. What is the Android Keystore? — Secure storage for cryptographic keys.
6. What is WebView risk? — JS injection and insecure bridges.
7. What is the `android:debuggable` flag risk? — Enables debugging and data access.
8. What is a content provider? — Data sharing component with permissions.
9. What is app signing? — Cryptographic signature proving app integrity.
10. What is Play Integrity API? — Attestation for device and app integrity.
11. What is ADB and why is it risky in production? — Debug bridge access.
12. What is `FileProvider`? — Safe file sharing mechanism.
13. Why are hardcoded API keys dangerous? — Extractable from APK.
14. What is backup risk? — Sensitive data included in backups.
15. What is `networkSecurityConfig`? — TLS and trust settings for the app.
