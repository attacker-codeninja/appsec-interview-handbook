# Serverless Security — Mid-Senior

1. How do you design least privilege IAM policies? — Start narrow and monitor usage.
2. How do you secure event sources? — Validate signatures and schema.
3. How do you handle secret management? — Use secret managers with rotation.
4. How do you prevent data exfiltration? — VPC egress controls and logs.
5. How do you secure API Gateway? — Auth, throttling, and WAF.
6. How do you protect against SSRF in serverless? — Egress allowlists.
7. How do you monitor serverless security? — Structured logs and alerts.
8. How do you manage dependency risk? — SCA and image scanning for containers.
9. How do you handle multi-tenant serverless? — Tenant isolation and scoped tokens.
10. How do you secure function-to-function calls? — Signed requests or mTLS.
11. How do you manage code integrity? — Signed deployments and CI policies.
12. How do you handle secrets in env vars? — Avoid; fetch at runtime.
13. How do you handle large payloads securely? — Use object storage with signed URLs.
14. How do you secure state machines? — IAM and validation at each step.
15. How do you handle PII in logs? — Redaction and minimized data.
