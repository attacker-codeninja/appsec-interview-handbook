# Infrastructure and Cloud Security (General) — Junior

1. What is cloud IAM? — Identity and access management in cloud.
2. What is least privilege? — Minimal permissions required.
3. What is a security group? — Virtual firewall for instances.
4. What is a public bucket risk? — Data exposure.
5. What is a VPC? — Isolated virtual network.
6. What is a shared responsibility model? — Provider vs customer security.
7. What is encryption at rest? — Protecting stored data.
8. What is encryption in transit? — TLS for data transfer.
9. What is a cloud KMS? — Key management service.
10. What is an IAM role? — Temporary permissions for services.
11. What is a bastion host? — Controlled access to private network.
12. What is a cloud audit log? — Record of API actions.
13. What is a CSPM tool? — Cloud security posture management.
14. What is a default VPC risk? — Broad network access.
15. What is a misconfigured firewall? — Opens unintended access.
