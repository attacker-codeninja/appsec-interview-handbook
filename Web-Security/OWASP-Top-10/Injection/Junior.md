# OWASP Top 10: Injection — Junior / Entry Level

1. What is SQL injection? — Injecting SQL through untrusted input.
2. What is NoSQL injection? — Injecting operators into NoSQL queries.
3. What is OS command injection? — Executing system commands via input.
4. What is LDAP injection? — Manipulating directory queries.
5. What is XPath injection? — Manipulating XML queries.
6. What is template injection (SSTI)? — Executing template expressions.
7. Why is string concatenation risky in queries? — It mixes code and data.
8. What is input validation? — Ensuring inputs match expected formats.
9. What is output encoding? — Escaping data in outputs.
10. What is a prepared statement? — Precompiled query with parameters.
11. Why are stored procedures not always safe? — They can still concatenate inputs.
12. What is command injection via file upload? — Passing filenames into shell.
13. What is log injection? — Inserting control characters into logs.
14. What is HTTP header injection? — Injecting newline into headers.
15. Why is least privilege for DB users important? — Limits impact of injection.
