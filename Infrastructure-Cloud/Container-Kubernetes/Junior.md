# Container and Kubernetes Security — Junior

1. What is a container image? — A packaged filesystem with app and deps.
2. What is Kubernetes? — Container orchestration platform.
3. What is a pod? — Smallest deployable unit in Kubernetes.
4. What is RBAC? — Role-based access control.
5. What is a namespace? — Logical isolation in Kubernetes.
6. What is a privileged container? — Container with elevated host access.
7. What is an admission controller? — Policy enforcement on API requests.
8. Why are mutable tags risky? — `latest` can change unexpectedly.
9. What is a service account? — Identity for pods.
10. What is a kubeconfig? — Cluster access config file.
11. What is a network policy? — Controls pod traffic.
12. What is a container escape? — Breaking isolation to host.
13. What is image signing? — Verifying image integrity.
14. What is a runtime policy? — Detects suspicious container behavior.
15. What is a base image? — Starting image for builds.
